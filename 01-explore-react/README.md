# Explore React

### 01. Initialize project with vite

```bash
npx create-vite@latest
```

```bash
npm create vite@latest
```

### 02. Install dependencies

```bash
npm install
```

### 03. Run project

```bash
npm run dev
```

### 04. Delete unnecessary files

- Delete `src/assets`
- Delete `src/main.css`
- Delete `src/App.css`
- Delete `src/index.css`
- Delete `src/logo.svg`
